Getting Started

  1. Create an HTTP connection named starship_default pointing to your source Airflow's Starship API (the instance you're migrating from).
  2. Choose a migration mode: Big-Bang migrates all DAGs at once; Incremental lets you pick DAGs by pattern.
  3. Monitor progress on the status page — it auto-refreshes while running.
  4. If something goes wrong, use Rollback to undo or Retry to re-run failed DAGs.
